## üìò Angular `@HostListener` ‚Äì Numbered Notes

1. `@HostListener` is an **Angular decorator**.

2. It is used to **listen to DOM events** on the **host element**.

3. When the specified event occurs, it **executes an event handler method**.

4. The **host element** is the HTML element on which the directive is applied.

5. Example of a host element:

   ```html
   <div appHighlight></div>
   ```

6. A custom directive can be created using Angular CLI:

   ```bash
   ng generate directive highlight
   ```

7. Angular CLI automatically:

   * Creates directive files
   * Declares the directive in `app.module.ts`

8. The directive selector is defined using the `@Directive` decorator:

   ```ts
   selector: '[appHighlight]'
   ```

9. The directive selector is used as an **attribute directive**.

10. `ElementRef` is used to get a reference to the **host DOM element**.

11. `element.nativeElement` gives access to the actual DOM element.

12. `Renderer2` is used to **safely manipulate the DOM**.

13. Direct DOM manipulation is avoided for:

    * Security
    * Platform independence
    * Server-side rendering compatibility

14. `@HostListener` is used to listen to events on the host element.

15. Syntax of `@HostListener`:

    ```ts
    @HostListener('eventName')
    methodName() {}
    ```

16. Commonly used DOM events:

    * `mouseenter`
    * `mouseout`
    * `click`
    * `keydown`

17. `mouseenter` event is used to apply a zoom/highlight effect.

18. CSS class is added using:

    ```ts
    this.renderer.addClass(this.element.nativeElement, 'highlight-product');
    ```

19. `mouseout` event is used to remove the zoom/highlight effect.

20. CSS class is removed using:

    ```ts
    this.renderer.removeClass(this.element.nativeElement, 'highlight-product');
    ```

21. Component-level CSS is **scoped** in Angular.

22. Styles must be defined in:

    * The same component‚Äôs CSS file
      **OR**
    * Global `styles.css`

23. Hovering over the product zooms it in.

24. Moving the mouse away zooms the product out.

25. `@HostListener` keeps event-handling logic **out of HTML templates**.
